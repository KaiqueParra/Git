Comando para usar o git

git init = ele incia um repositório na pasta atual git (master)

git add = ele manda os arquivos para area de staging area (prepara para o commit)

git status = verifica o estado atual dos arquivos (rastreados, não rastreados, modificados)

git commit -m "mensagem" = salva as alterações da staging area no repositório local

git branch -M "main" = para mudar a nomenclatura da branch principal de (master) para (main)

git remote add origin LINK DO REPOSITORIO = para criar a conexao do repositorio locar para o repositorio do github

git push -u origin main = para mandar os commits e arquivos da minha maquina para o repositorio do github

$ git checkout -b "" = parar criar uma nova Branch, tudo oque eu alterar na nova branch vai ser so alterada nela e nao na (main)
